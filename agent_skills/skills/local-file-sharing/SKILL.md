---
name: local-file-sharing
description: This skill is used to upload local files to users through links. Please use this skill when local file transfer is required.
---

If users of the front-end webpage need to obtain files generated during the operation of the intelligent assistant, they can use 'scripts/upload2hepai. py' to upload the files to the HepAI server and obtain the link to the files. **Attention**, the uploaded file size must be less than 10M. Check the file size before use.
