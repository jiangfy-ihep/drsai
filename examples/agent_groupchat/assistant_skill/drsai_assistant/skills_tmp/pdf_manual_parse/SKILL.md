# Guide

## 用途

应用于解析PDF手册、长文档等文件，并上传到RAGFlow知识库检索框架进行检索。

**注意**：

- 只适合解析手册、毕业论文、书籍等明确包含目录的文档，解析过程中出现相关问题要明确向用户说明。
- 用户解析的文件必须是HepAI文件系统的ID。如果不是，则提醒用户将必须先将文件上传到文件系统，输入文件的ID。
