# OpenDrSai-智能体框架内的子系统

在OpenDrSai的智能体框架内，主要有管理智能体状态的状态管理子系统、管理智能体配置的配置管理子系统，这将在后面的文档中进行进一步的介绍。

## 1.状态管理子系统

主要控制智能体实例状态的全局函数：

- `lazy_init`：初始化时，智能体内部需要执行的操作
- `close`：关闭智能体实例时，智能体内部需要执行的操作
- `pause`：暂停智能体时，智能体内部需要执行的操作
- `resume`：重启智能体时，智能体内部需要执行的操作
- `on_reset`：重置智能体时，智能体内部需要执行的操作
- `save_state`：保存智能体状态时，智能体内部需要执行的操作
- `load_state`：加载智能体状态时，智能体内部需要执行的操作

## 2.配置管理子系统

导出和加载智能体实例状态的全局函数：

- `_to_config`：将智能体定义参数、各个组件参数、变量参数等导出为json参数
- `_from_config`：通过json参数重构智能体实例
